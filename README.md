# 🚀 NeXRecipt — AI Powered Android App

A modern Android application built with Jetpack Compose, Kotlin, and AI Studio integration.  
Designed for productivity, smart automation, and next-generation AI experiences.

---

## 📱 Features

- ⚡ Jetpack Compose UI (Modern Android UI)
- 🤖 Gemini AI integration
- 📦 Room Database support
- 🌐 Retrofit networking
- 🔐 Secure API key handling via `.env`
- 📊 Clean MVVM architecture
- 🚀 GitHub Actions CI/CD support

---

## 🛠️ Tech Stack

- Kotlin (Latest Stable)
- Jetpack Compose
- KSP (Kotlin Symbol Processing)
- Room Database
- Retrofit + OkHttp
- Moshi JSON
- Coroutines + Flow
- Firebase (optional services)
- Gradle 9 + AGP 9
